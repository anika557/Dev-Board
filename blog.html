Question-1: What are the different ways to select an element in the DOM?

We can use getElementById for a single element by its ID, getElementsByClassName for elements by class, or getElementsByTagName for elements by tag. For more flexibility, querySelector and querySelectorAll allow CSS-style selectors, while getElementsByName selects elements by their name attribute.

Question-2: What is the difference between innerHTML, innerText, and textContent ?

innerHTML, innerText, and textContent are used to access or update content in the DOM, but they differ in behavior. innerHTML retrieves or sets the HTML content inside an element, including tags. innerText returns only the visible text, ignoring hidden elements and respecting CSS styling

Question-3: What is event delegation in the DOM?

Event delegation is a technique in the DOM where we attach a single event listener to a parent element instead of adding multiple listeners to individual child elements. This approach leverages event bubbling, where events triggered on child elements propagate up to the parent..

Question-5: How do you create, add, and remove elements using JavaScript?

To create an element, use document.createElement(). To add it to the DOM, use methods like appendChild() or insertBefore(). To remove an element, use removeChild() on its parent or the modern element.remove(). These methods allow dynamic manipulation of the DOM.
